Git & GitHub - Partie 2 Activité

Exercice ayant pour but de mettre en avant les connaissances acquises depuis les deux premiers chapitres.

Notamment :
- La création d'un fichier README.txt qui présente le projet et ses étapes
- La créations de 2 autres fichiers
- La réaslisation d'au moins 4 commits distincts

Ceci fait, le repository devra être envoyé vers GitHub, l'url ainsi généré sera indiqué dans le fichier.txt prévu à cette effet et renvoyé dans un fichier compressé pour cloturer l'exercice.
